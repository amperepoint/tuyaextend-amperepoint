# TuyaExtend AmperePoint

AmperePoint Home Assistant helper integration plus EVSE profiles, dashboards,
DP maps and sanitized Tuya observations.

Before installing TuyaExtend AmperePoint, initialize the official Home Assistant
Tuya integration and confirm that the charger is visible in Home Assistant.

See `INSTALL.md`, `README.md` and `amperepoint/README.md`.
